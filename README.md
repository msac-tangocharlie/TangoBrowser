# Features
- History
- Pin Pages
- Incognito Mode
- Panel for shortcuts
- DevTools
- Find in Page
- Color Customization
- Personal Profile
- Restored Session
- Supports New Websites
- Dark Mode
- Site Loading Time
- Download and Save As option
